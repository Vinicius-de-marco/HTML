Links

Aqui vou mostrar para o que a função link serve

A função link serve para quando clicar em um link, vai ser redirecionado para a pagina de link que vinculou

Por exemplo:

Isso é um link (clique nele)

Para usar essa função, eu usei o comando

"a href="https://www.instagram.com/vini.demarco/">Isso é um link (clique nele) /a"
